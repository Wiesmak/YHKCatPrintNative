Detailed Description

template<std::derived_from< IDevice > TDevice>
class yhkcatprint::AdapterWin32< TDevice >

Win32 implementation of IAdapter interface.

This class implements the IAdapter interface using the Windows Bluetooth API. It provides functionality to interact with Bluetooth adapters on Windows platforms.

Template Parameters
TDeviceType of Bluetooth device to use, must derive from IDevice.
