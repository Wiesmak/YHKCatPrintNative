Detailed Description

Abstract interface for RFCOMM socket communication.

Implementations of this interface provide a byte-oriented, stream-like API over an RFCOMM connection.

Note
Unless specified otherwise by an implementation, methods may block.
Implementations should document their thread-safety guarantees.
